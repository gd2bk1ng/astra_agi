# Astra Learning Crate

## Overview

Handles adaptive learning algorithms, reinforcement learning, and model updates.

## Main Modules & Functions

- `learn_from_feedback(feedback: Feedback)`  
  Incorporate feedback into learning.

- `update_models()`  
  Refine internal models.

- `predict_outcomes(state: &State) -> Prediction`  
  Forecast future states.

## Usage

Enables Astra to improve and adapt continuously.
